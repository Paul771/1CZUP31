Приемы на работу, переводы, увольнения

Журнал предназначен для работы с документами:

  • Прием на работу
  • Кадровый перевод
  • Перемещение в другое подразделение
  • Перевод к другому работодателю
  • Увольнение
  • Отпуск по уходу за ребенком
  • Изменение условий оплаты отпуска по уходу за ребенком
  • Возврат из отпуска по уходу

Если в программе работают пользователи с ограниченными правами, то не до конца оформленные (не утвержденные) документы в списках выделяются полужирным шрифтом. Утвердить документы можно непосредственно из списка, не открывая их, с помощью кнопки Утвердить.
