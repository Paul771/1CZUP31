Правило заполнения реквизита объекта 1С:Документооборота может задавать один из трех способов заполнения:

  • Из реквизита этой конфигурации. Этот способ означает, что реквизит объекта 1С:Документооборота будет заполнен согласно значению выбранного реквизита объекта интегрируемой конфигурации.
  • Из указанного значения. Этот способ означает, что реквизит объекта 1С:Документооборота будет заполнен фиксированным значением.
  • Из выражения на встроенном языке. Этот способ означает, что реквизит объекта 1С:Документооборота будет заполнен из результата выражения на встроенном языке 1С:Предприятия. При выборе этого способа требуется ввести выражение, результат которого будет использован для заполнения. В выражение могут входить любые операции, в том числе и вызовы экспортных процедур и функций. Результат выражения должен быть присвоен переменной Результат. Переменная Источник содержит объект интегрируемой конфигурации, на основании которого создается объект 1С:Документооборота.

В распространенном случае реквизит объекта 1С:Документооборота ссылочного типа заполняется согласно значению реквизита интегрируемой конфигурации, который тоже имеет ссылочный тип. Например, реквизит Отправитель входящего документа может заполняться согласно значению реквизита Контрагент документа Счет на оплату. Поиск соответствия выполняется в таком порядке:

  1. По связи, установленной ранее при интерактивном создании на основании или при выборе соответствия вручную на закладке Документооборот.
  2. По уникальному идентификатору. Соответствие будет найдено, если используется синхронизация данных, и контрагент 1С:Документооборота создан загрузкой контрагента из интегрируемой конфигурации или наоборот.
  3. Для справочников Контрагенты – по полям поиска в порядке убывания приоритета: ИНН и КПП; ИНН и наименование; наименование. Для остальных справочников –по наименованию.

В отсутствие соответствия создается новый объект.

ВАЖНО! При поиске соответствий по полям поиска не используются правила заполнения реквизитов, даже если они заданы соответствующим правилом интеграции.
