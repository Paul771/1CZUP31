Файл загружается с помощью кнопки «Загрузить». В поле «Наименование» определяется представление файла в качестве страницы теории. По умолчанию, наименование – это имя загружаемого файла.

Определены следующие типы загружаемых файлов:

  • По умолчанию. Файл отображается на странице курса в виде ссылки, при переходе по которой файл будет открыт. Если браузер не умеет отображать файлы данного формата, то он предложит сохранить его или открыть в приложении, работающем с файлами данного формата.
  • Изображение. Скриншоты, картинки и другие изображения различных форматов, которые поддерживаются браузерами. Например, JPG, PNG, GIF. Для изображения можно задать ширину, высоту и положение выравнивания на странице.
  • Видео. Видеофайлы, которые можно просматривать непосредственно на странице. В некоторых случаях для воспроизведения файла потребуется установить нужный плагин. Рекомендуется размещать видео в формате mp4.
  • Аудио. Данный тип файла позволяет прослушивать аудиозаписи. Рекомендуется использовать файлы в формате mp3.

Тип файла определяется системой автоматически после его загрузки, поэтому вручную тип файла устанавливать не нужно. Если программа не смогла определить тип файла при загрузке, то файлу устанавливается тип «По умолчанию». В зависимости от флажка «Открывать внешним приложением» такой файл может открываться приложением, предназначенным для его просмотра, или демонстрироваться в браузере.
