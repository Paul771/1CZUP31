Форма заполнения респондентов

Форма предназначена для составления списка респондентов, участвующих в мероприятии оценки персонала.
Предназначена для заполнения списка сотрудников при индивидуальном набор респондентов для каждого сотрудника

Позволяет подобрать для каждого сотрудника состав респондентов, состоящий из его руководителя, коллег, подчиненных. Для заполнения списка респондентов коллегами сотрудника (сотрудниками, работающими с ним в одном подразделении) и им самим необходимо поставить отметку о необходимости их включения. Роль сотрудника при этом заполнится, как самооценка; подразумевается, что сам сотрудник добавляется в респонденты, чтобы сравнить оценки респондентов с оценкой себя самим сотрудником.
На список респондентов можно также ввести ограничения.
