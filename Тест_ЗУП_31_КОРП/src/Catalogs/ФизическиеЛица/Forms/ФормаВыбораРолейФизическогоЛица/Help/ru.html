Форма выбора ролей физического лица

В форме можно указать, какие взаимоотношения имеет лицо с организацией. Тогда при выборе лиц в документах, предназначенных, например, для регистрации получения доходов такими лицами (в указанной организации), по умолчанию будет предлагаться, в том числе, и это лицо. Если же в таких документах лицо было выбрано не из списка лиц по умолчанию, а из всего списка физических лиц, то признак такому лицу устанавливается автоматически.

Приведенные роли определяют доступность лица для выбора по умолчанию в следующих документах:

Акционер – Дивиденды;
Получает доход по прочим договорам – Регистрация прочих доходов;
Бывший сотрудник – Выплата бывшим сотрудникам;
Раздатчик зарплаты – Ведомость выплаты через раздатчика.

Если предприятие состоит из нескольких организаций, то можно указать характер взаимоотношений для каждой организации отдельно.
