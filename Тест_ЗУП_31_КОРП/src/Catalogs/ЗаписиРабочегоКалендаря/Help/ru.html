Рабочий календарь пользователя предназначен для планирования времени. В нём можно вводить различные записи и отмечать время, доступное для планирования другим сотрудникам. Для включения календаря необходимо установить настройку программы Использовать рабочий календарь.

Варианты просмотра календаря

Для работы с календарем пользователя предназначена команда Мой календарь в панели навигации рабочего стола и в разделе Совместная работа. Мой календарь можно просматривать в различных видах: День, Неделя, Две недели, Месяц, План дня.

Виды День, Неделя, Две недели отображают события каждого дня по часам. Данные режимы удобны тем, что позволяют оценить доступность времени для каждого часа каждого дня. Текущий день выделяется специальным образом - синий заголовок и светло-желтый фон.

Вид Месяц отображает календарь на месяц в целом. Если на какой-либо день запланировано слишком много событий, будет отображаться надпись с общим числом событий, например "Всего 5", при нажатии на которую можно будет просмотреть список событий данного дня.

В виде План дня отображаются записи календаря в виде таблицы, с группировкой по дням. События отображаются на 9 дней, начиная с позавчерашнего дня.

Как создать запись календаря

Для создания новой записи календаря выделите нужное время мышкой и нажмите Ctrl+N или выполните команду Создать - Запись календаря.

Подбор времени

Для записей календаря можно устанавливать различные цвета.

Записи календаря на весь день

Для записей календаря можно указывать не конкретное время в пределах дня, а в целом весь день. Для этого предназначен специальный флаг Весь день. Записи календаря на весь день отображаются в календаре специальным образом. В виде просмотра План дня события на весь день выводятся первыми и вместо времени у них отображается надпись "Весь день". В видах просмотра День, Неделя, Две Недели события всего дня отображаются в верхней части календаря, возле даты. Управлять свернутостью данных событий можно с помощью двойного нажатия в области даты календаря или с помощью команды Свернуть события всего дня.

Создание записей календаря на основании предметов

Записи календаря можно создавать на основании различных предметов, например на основании внутренних документов, писем, контрагентов. При этом предмет, на основании которого создана запись календаря, отображается в нижней части карточки записи в виде гиперссылки.

Область чтения

Для быстрого просмотра в календаре предусмотрена область чтения (отключена по умолчанию). Настройки области чтения независимы для различных отображений календаря.

Печать

Записи календаря и различные виды календарей можно вывести на печать, для этого предназначены специальные команды печати. На печать можно вывести как текущий вид календаря, так и отдельные записи календаря. Для вывода на печать текущего вида календаря следует выполнить команду Печать – Календарь. Для вывода нескольких записей календаря на печать следует выделить данные записи и выполнить команду Печать - События.

Форма рабочего стола

На рабочем столе доступна форма, события в которой отображаются как в режиме просмотра План дня, но только на 3 дня, начиная с сегодняшнего дня.

Напоминания

Для записей календаря можно устанавливать напоминания. Для этого необходимо включить настройку программы Использовать напоминания. После этого станет доступна настройка напоминаний в карточке записи календаря. В персональных настройках можно указать срок напоминания по умолчанию и необходимость автоматической установки напоминаний.

Календари других пользователей

С помощью команды Все календари в разделе Совместная работа можно просматривать календари других пользователей. Конкретные записи будут отображаться только в том случае, если они доступны по правам доступа. Если же они не доступны, то будет отображаться только состояние занятости на данное время. Для записей календаря права распространяются по руководителям и делегатам.

Можно вносить записи в календари других пользователей. Тогда данная запись будет доступна автору и пользователю, который внёс данную запись. Вносить записи в календари других пользователей можно только в состоянии Под вопросом. Для внесения записи в календарь другого пользователя необходимо в карточке записи календаря выполнить команду Отображать поле Пользователь и поменять пользователя в появившемся поле. Если пользователь и автор записи различные, то автор будет отображаться в карточке записи.

Для событий доступно три состояния. В зависимости от состояния, может изменяться занятость пользователя на данное время:

• Принято – время считается занятым;

• Под вопросом – время считается под вопросом;

• Отклонено – время считается доступным.

Настройка доступного времени по расписанию

С помощью команды Настроить доступное время в подменю Все действия можно указать, в какое время пользователь будет считаться доступным в течении недели.

Настройка доступного времени

Пользователь может отмечать доступность отдельного времени как занятого, доступного или под вопросом. Для этого можно выделить область в календаре и выполнить команду контекстного меню Пометить как доступное / недоступное. Тогда доступность в выбранного времени изменится на противоположную. Можно также использовать команду Доступность для более точной настройки доступности с указанием конкретного времени и состояния занятости.

Настройка повторения записей календаря

Для записей календаря можно указывать настройки повторения. После этого запись с заданными настройками повторения будет отображаться в календаре. Отдельный элемент повторения можно изменить или удалить. При изменении всех записей повторения данные в отдельных элементах повторения будут заменены на новые общие настройки.

Мероприятия в календаре

При одновременном включении настроек программ Использовать рабочий календарь и Управление мероприятиями будет произведено начальное заполнение календаря данными о мероприятиях. После этого мероприятия будут отображаются в календарях участников. Доступ к данным записям календаря имеют все участники мероприятия. Изменяя состояния записи календаря, связанной с мероприятием, пользователь будет менять состояние приглашения на мероприятие. При внесении изменений в мероприятия, данные изменения отображаются в записях календаря пользователей.

Подбор времени мероприятия

В карточке мероприятия добавлены команды Подобрать время и Проверить доступность участников. Данные команды предназначены для контроля того, что выбранное время является доступным для участников. При рассылке приглашений и при принятии приглашения проверка доступности выполняется автоматически.

Команда Проверить доступность участников позволяет проверить, доступны или заняты во время мероприятия. Если окажется, что не все пользователи доступны, то будет выведено окно с состояниями пользователей и возможностью перейти к подбору времени.

Форма подбора времени участников предназначена для выбора времени мероприятия, которое удобно для всех участников. Программа даёт рекомендации о времени проведения мероприятия, выводя их в Подходящее время. Условия, по которым ищется свободное время, отображается справа. С помощью гиперссылки Подробнее можно настроить условия подбора - изменить время, даты, состав участников поиска. На форме так же присутствует отображение занятости участников подбора по часам в пределах дня.
