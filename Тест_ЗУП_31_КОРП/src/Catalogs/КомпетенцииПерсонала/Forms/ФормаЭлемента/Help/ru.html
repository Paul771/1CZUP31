Форма элемента

Возможно ввести описание компетенции и вопросы, по которым будет оцениваться компетенция.
Вопросы добавляются из справочника Вопросы для анкетирования. Необходимо выбирать только вопросы с типом ответа: Выбор нескольких вариантов ответа из предложенных. Для этого типа необходимо также ввести варианты ответов.
Оценки ответов можно уточнить по одноименной кнопке.
