Статьи расходов

Справочник "Прочие расходы" предназначен для ведения списка произвольных прочих расходов, которые не связаны с объектами программы (организацией, подразделением, заказами и т.д). Список прочих расходов определяется для той статьи расходов , в которой в качестве аналитики указано "Прочие расходы".

Для каждой статьи расходов, у которой установлен тип аналитики "Прочие расходы" можно определить свой список прочих расходов, который будет принадлежать только этой статье расходов.

При указании статьи расходов в документах в качестве аналитики нужно будет выбрать наименование прочего расхода из справочника "Прочие расходы. Это позволит провести более детальный анализ расходов торгового предприятия в отчете "Анализ доходов и расходов" и получить правильный финансовый результат в отчете "Анализ финансовых результатов".
