Подразделения

Каждая из организаций (входящих в предприятие) может иметь сложную структуру, состоять из различных подразделений, в том числе обособленных с точки зрения НК РФ (зарегистрированных в отдельных ИФНС).

Описываемые в программе подразделения используются, в первую очередь, в целях ведения кадрового учета: для каждого сотрудника обязательно указывается подразделение, в которое он принимается, в которое перемещается и т.п. Подразделения указываются и в штатном расписании, если оно ведется в программе. Во-вторых, зарплата сотрудников также регистрируется (учитывается) с привязкой (в разрезе) подразделений.
Соответственно, подразделения указывают в печатных формах, отчетах, формируемых по результатам ввода данных и проведения расчетов.

Для каждого подразделения в программе можно описать особенности, касающиеся отражения доходов его сотрудников бухгалтерском учете, места выплаты зарплаты его сотрудникам, территориальных условий для отчетности в ПФР и т.п.

При описании структуры организации в программе необходимо учитывать, что она считается единой для всех перечисленных целей: кадрового учета, расчета зарплаты, отражения в учете, формирования отчетности и т.п.

Структура подразделений может быть описана иерархически, т.е. некоторые подразделения могут являться "вышестоящими" для других. Это предоставляет дополнительные удобства при группировке данных в различных отчетах.

Список подразделений

Помимо непосредственно создания подразделений в списке можно настроить их порядок и затем использовать его для сортировки в отчетах, где фигурируют подразделения. Порядок настраивается с помощью стрелок в панели над списком.

Если в программе ведется штатное расписание, то по умолчанию в списке подразделений не показываются "недействующие" подразделения.
К таким подразделениям относятся:
• "новые", т.е. для которых еще не указано, что они сформированы (включены в штатное расписание);
• расформированные, т.е. для которых указано, что подразделение расформировано и больше не используется.
При этом в форме доступны флаги, установка которых позволяет показать в списке такие подразделения. При их установке "новые" подразделения отмечаются жирным шрифтом, а расформированные – зачеркнутым.
