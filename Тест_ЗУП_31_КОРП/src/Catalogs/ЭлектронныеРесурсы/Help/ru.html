Набор файлов или ресурс представляет собой контейнер для файлов. При отображении ресурса отображаются все его файлы, если в настройках файла не указано «Скрывать» и не указан файл запуска. Ресурс имеет стандартный для всех объектов набор реквизитов.

Следует помнить, что перед отображением ресурса или хотя бы одного файла ресурса происходит кэширование всех его файлов на локальном компьютере пользователя (при просмотре неопубликованных электронных ресурсов). По этой причине во избежание замедления работы программы в один ресурс следует включать только те файлы, которые предполагается использовать вместе. Если  файл будет использован многократно в различных объектах, то его следует поместить в отдельный ресурс.
