Фасеты позволяют задать несколько вариантов записи одного вопроса. Фасеты повышают валидность тестов за счет того, что они помогают снизить искажение результатов из-за списывания. Так же фасеты значительно снижают трудоемкость разработки параллельных вариантов тестов.

Фасет должен обязательно принадлежать группе. В группу следует объединять те фасеты, сочетания значений которых при подстановке в текст вопроса дают одинаковый ответ. Каждый фасет состоит из набора значений, которые подставляются в заданное место текста вопроса. Место в тексте вопроса, куда вставляются значения фасета, задается при помощи обрамления номера фразы в двойные фигурные скобки, например {{2}}.
 
После того, как будут заданы необходимые группы фасетов, следует указать, для какой группы какой вариант ответа будет верным. Для этого переходим на вкладку "Варианты ответов", делаем для варианта пометку "Верный" и указываем нужную группу фасетов в колонке "Верные фасеты".
