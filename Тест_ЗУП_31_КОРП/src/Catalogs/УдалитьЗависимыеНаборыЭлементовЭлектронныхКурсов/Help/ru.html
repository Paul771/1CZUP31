Если файлы ресурса используют файлы других ресурсов, то их можно указать в справочнике "Зависимые ресурсы". Например, если ресурс содержит файл MS Word, который имеет ссылки на другие файлы, то для обеспечения работы этих ссылок необходимо на вкладке "Зависимые ресурсы" указать необходимые для работы ссылок ресурсы.
