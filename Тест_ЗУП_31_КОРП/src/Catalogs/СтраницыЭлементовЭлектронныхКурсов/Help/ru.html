Подборка состоит из последовательности групп элементов. Группы элементов образуют страницы подборки. Каждая группа может содержать один или несколько элементов. Элементы каждой группы также можно объединять в разделы. Таким образом, подборка позволяет создать двухуровневую информационную структуру, которая будет сочетать в себе удобную навигацию и разнообразие возможностей для представления информации.
