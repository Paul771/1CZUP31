Глоссарий выполняет в электронном курсе функцию словаря используемых терминов. Для электронного курса можно создать новый глоссарий или воспользоваться уже существующим. При создании нового глоссария в него необходимо на закладке «Содержание» добавить список терминов и установить настройки отображения на закладке «Настройка».

1. На закладку «Содержание» кнопкой «Добавить из ранее созданных» можно добавить термины, уже существующие в информационной базе. Новый термин создается с помощью кнопки «Создать термин».

2. На закладке «Настройка» возможна установка двух флажков.

При установке флажка «Показывать с определением» термины глоссария будут отображаться сразу с определением. При снятом флажке выводится список наименований терминов, каждый из которых представлен ссылкой. Чтобы увидеть определение термина, нужно выбрать соответствующую ссылку.

Если установлен флажок «Показывать алфавит», то в верхней части страницы будет отображаться алфавит. Те буквы алфавита, на которые начинаются термины в глоссарии, активны и выделены цветом. При нажатии на такую букву открывается список терминов, начинающихся на эту букву. При снятом флажке термины отображаются в виде списка, упорядоченного по алфавиту.
