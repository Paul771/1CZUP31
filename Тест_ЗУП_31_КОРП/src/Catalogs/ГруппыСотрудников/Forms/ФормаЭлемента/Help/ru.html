Группа сотрудников

При большом количестве сотрудников может требоваться различать их по тем или иным признакам. Такое разделение удобно как для визуального восприятия списка сотрудников, так и при оформлении списочных документов или построении аналитических отчетов только по определенным сотрудникам.

В программе имеется возможность создания произвольных групп сотрудников в этом справочнике. При этом:
- одного сотрудника можно отнести сразу к нескольким группам;
- можно настроить автоматическое отнесение сотрудников к группам по заданным условиям.

Также в программе имеются и другие подобные возможности:
- визуальная группировка списка сотрудников по их подразделениям;
- помещение уволенных сотрудников «в архив» и скрытие таких сотрудников из списка;
- добавление произвольных реквизитов для сотрудников и использование их при построении отчетов.

Настройка группы

Для простой группы достаточно задать произвольное наименование и перечислить относящихся к ней сотрудников.

Но можно создать группу не только для отнесения сотрудников к ней вручную, но и с автоматическим заполнением. Для этого достаточно установить флажок Формировать автоматически и настроить условия (кнопка Настроить критерии отбора), по которым сотрудники будут относиться к этой группе.

Состав автоматически формируемых групп не фиксируется на момент их создания. Он зависит от того, какие сотрудники в настоящий момент подходят под настроенный для группы отбор.
При повторном открытии автоматически формируемой группы посмотреть список входящих в нее сотрудников можно по кнопке Сформировать список отбираемых сотрудников.
