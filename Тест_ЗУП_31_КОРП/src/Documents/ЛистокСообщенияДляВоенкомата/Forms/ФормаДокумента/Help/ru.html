Листок сообщения об изменениях

Документ предназначен для формирования и печати "Листка сообщения об изменениях сведений о гражданах, состоящих на воинском учете" (приложение 13 к Методическим рекомендациям Генштаба ВС РФ) в случае изменения личных данных граждан, состоящих на воинском учете.

Проанализировать изменение личных данных можно с помощью аналитического отчета Анализ изменений личных данных сотрудников. По результатам анализа можно ввести этот документ.

Регистрация в программе листков сообщения позволяет автоматически заполнить отчет Сведения об изменениях для военкомата, который следует представлять в военкоматы по гражданам, самостоятельно не представившим листки в военкомат.

Работа с документом

Документ регистрируется со статусом листок не выдан. Его следует распечатать для передачи сотруднику (кнопка Напечатать). При этом перед печатью на закладке Сведения о сотруднике можно проверить личные данные, которые попадут в листок.

Если своевременно передать сотруднику распечатанный листок не удалось, необходимо указать причину в одноименном поле. Эта причина впоследствии будет записана в графу 7 (Примечание) отчета об изменениях для военкомата.

При этом в поле Содержание изменений для формы "Сведения об изменениях для военкомата" следует в произвольном виде описать содержание изменений. Эта информация будет записана в графу 6 отчета (Примечание).

При вручении сотруднику листка на руки необходимо изменить статус документа на листок выдан и указать дату вручения.

Печатная форма листка включает в себя корешок, в котором в военкомате ставится отметка о представлении его сотрудником. При получении корешка с печатью военкомата следует установить флажок Корешок листка получен.

Данные о сотрудниках, не предъявивших в установленный срок корешок листка сообщения, включатся в отчет Сведения об изменениях для военкомата. В таких листках также следует заполнить поле Содержание изменений для формы "Сведения об изменениях для военкомата" для заполнения графы 6 отчета. Графа 7 будет заполнена датой вручения листка.
