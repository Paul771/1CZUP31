Платежное поручение

Документ предназначен для подготовки реестра перечислений в банк в электронном виде в рамках зарплатного проекта. Он позволяет сформировать один файл для нескольких ведомостей, оплаченных одним платежным поручением.
Файл реестра перечислений по одной ведомости можно сформировать непосредственно из нее, не прибегая к вводу платежного поручения.
Работа с документом предполагается из рабочего места Обмен с банками по зарплатным проектам.
