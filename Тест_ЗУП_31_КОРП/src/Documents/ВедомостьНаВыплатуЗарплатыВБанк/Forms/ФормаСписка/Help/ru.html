Ведомости на выплату зарплаты через банк

Документ предназначен для регистрации факта передачи в банк информации о суммах, которые должны быть перечислены сотрудникам на карточки, открытые в рамках зарплатного проекта. Поддерживается формирование печатной формы реестра, а также формирование реестра в электронном виде.

Для регистрации перечислений на произвольные банковские счета сотрудников (открытые ими самостоятельно, а не в рамках зарплатного проекта) предназначен документ Ведомость перечислений на счета.

Список ведомостей

Если используется электронный обмен с банками, то выгрузить ведомость в банк в файл или включить в платежное поручение можно непосредственно из списка.
Однако для повышения удобства работы по обмену с банком предусмотрено рабочее место Обмен с банками по зарплатным проектам.
