Прием на работу списком

Документ предназначен для регистрации приема на работу нескольких сотрудников и для оформления соответствующего приказа о приеме по унифицированной форме Т-1а.
Работа с каждой строкой документа аналогична работе с документом Прием на работу, который предназначен для оформления только одного сотрудника.

Для оформления в программе приема на работу всех сотрудников, которые уже работают на предприятии на момент начала использования программы, предназначен специальный документ Данные на начало эксплуатации.
