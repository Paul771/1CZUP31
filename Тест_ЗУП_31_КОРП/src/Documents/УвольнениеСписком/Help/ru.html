Увольнение списком

Документ предназначен для регистрации увольнения нескольких сотрудников и для оформления соответствующего приказа по унифицированной форме Т-8а.
Работа с каждой строкой документа аналогична работе с документом Увольнение, который предназначен для оформления только одного сотрудника.
