Кадровый перевод списком

Документ предназначен для отражения в программе изменения (в том числе временного) условий работы нескольких сотрудников и печати соответствующего приказа по форме Т-5а.
Работа с каждой строкой документа аналогична работе с документом Кадровый перевод, который предназначен для оформления только одного сотрудника.
