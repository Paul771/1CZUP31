Входящая справка о заработке для расчета пособий

Документ предназначен для регистрации в программе данных из справки о заработке для расчета пособий, предоставленной сотрудником в соответствии с законодательством. Зарегистрированные документом данные могут учитываться при расчете среднего заработка для пособий ФСС.
