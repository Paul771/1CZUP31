Справка для расчета пособий (входящая)

Документ предназначен для регистрации в программе данных из справки о заработке для расчета пособий, предоставленной сотрудником в соответствии с законодательством. Зарегистрированные документом данные могут учитываться при расчете среднего заработка для пособий ФСС.

Заполнение документа

Документ заполняется в соответствии с предоставленной сотрудником справке.
Предыдущий страхователь, который выдал справку, выбирается из справочника Работодатели, куда его следует предварительно занести. При последующем расчете среднего заработка (с учетом данных из справки) будет отображаться название страхователя, данные которого учитываются.
Дополнительно можно указать период работы у этого страхователя в информационных целях.
