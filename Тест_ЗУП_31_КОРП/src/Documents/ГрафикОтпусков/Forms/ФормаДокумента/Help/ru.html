График отпусков

Документ предназначен для планирования графика отпусков и печати унифицированной формы Т-7.
Поддерживается перенос ранее запланированного отпуска с помощью документа Перенос отпуска.

Заполнение и печать документа

Дата документа и его номер используются при формировании печатной формы Т-7.

Ежегодные отпуска, которые требуется запланировать, перечисляются в таблице. На каждый вид отпуска вводится отдельная строка. В ней указывается сотрудник, период отпуска и его вид. Количество дней определяется автоматически. Подробнее о заполнении условий предоставления отпусков см. Отпуск.

После ввода всех отпусков в таблицу можно сформировать график отпусков по форме Т-7. Расшифровка подписи руководителя кадровой службы доступна в документе, по умолчанию она заполняется в соответствии с указанными в карточке организации данными.

По ссылке Этот документ оформлен правильно? можно проверить соответствие запланированных отпусков правам сотрудников на отпуск согласно учету остатков отпусков.

Непосредственно из графика можно оформить документы Отпуск в соответствии с запланированными условиями: по ссылке Оформить отпуск из строки таблицы можно оформить указанный в этой строке отпуск.

После утверждения графика отпусков может потребоваться перенести запланированный отпуск по просьбе сотрудника или по другим причинам. В программе такой перенос можно оформить документом Перенос отпуска, который можно ввести непосредственно из строки с ранее запланированным отпуском по ссылке Оформить перенос.
Для отпусков, на которые был оформлен перенос, в графике отпусков Т-7 заполняются предусмотренные для этого колонки.
