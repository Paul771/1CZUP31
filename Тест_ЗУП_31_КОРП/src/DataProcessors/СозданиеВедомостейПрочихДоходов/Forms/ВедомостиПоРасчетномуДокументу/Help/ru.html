Создание ведомостей на выплату зарплаты

В таблице автоматически отображается, какие ведомости могут быть созданы по этому документу, с учетом мест выплаты сотрудников. Каждую из "предполагаемых" ведомостей из этого списка можно открыть, посмотреть, при необходимости отредактировать в отдельном окне по кнопке Изменить.
Можно изменить параметры формирования ведомостей: например, указать, чтобы ведомости формировались отдельно по каждому подразделению, начисления по которому зарегистрированы в документе. При любом изменении параметров список ведомостей переформировывается автоматически. Вручную его можно обновить по кнопке Создать ведомости. После того как состав ведомостей подобран, по кнопке Провести и закрыть производится создание и регистрация оплаты ведомостей.
