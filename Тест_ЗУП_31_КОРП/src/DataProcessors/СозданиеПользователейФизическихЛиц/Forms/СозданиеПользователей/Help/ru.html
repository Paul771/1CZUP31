Создание пользователей для сотрудников

Рабочее место является инструментом регистрации в системе учетных записей пользователей, являющихся сотрудниками, для возможности использования самообслуживания.

Регистрация учетной записи осуществляется по команде Создать пользователя для выбранного из списка сотрудника, пока не имеющего учетной записи.

При создании пользователей учитываются настройки, заданные по соответствующей ссылке. Настройки созданного пользователя можно посмотреть по ссылке с именем пользователя справа, по этой же ссылке задается пароль доступа к системе.
Чтобы запретить вход в систему какому-либо пользователю, необходимо нажать Запретить вход в программу. Сотрудники, вход которым запрещен, будут отображаться в списке зачеркнутыми. Если пользователю нужно восстановить доступ, это можно сделать по кнопке Разрешить вход в программу.

Рассылка уведомлений

Для сотрудников, имеющих доступ к электронной почте, поддерживается рассылка уведомлений. Настройку уведомлений можно произвести по одноименной ссылке.

Указать адрес электронной почты возможно в Личных данных сотрудника.

По команде Отправить уведомление происходит отправка сообщения на почту сотрудника.
