Настройки пользователей

  • Аутентификация 1С:Предприятия - вход пользователя в систему будет осуществлять по логину и паролю для 1С:Предприятия
  • Показывать в списке выбора - определяет, показывать ли данного пользователя в списке для выбора при старте системы,
  • Пользователям запрещено изменять пароль - определяет возможность пользователя изменять свой пароль,
  • Аутентификация по протоколу OpenID- включите флажок, чтобы разрешить пользователю использовать единую учетную запись по протоколу OpenID. Данная функция используется для работы в веб-клиенте 1С:Предприятия. Для корректной работы программы в этом случае должен быть также включен флажок Аутентификация 1С: Предприятия.
  • Аутентификация операционной системы - для того чтобы упростить вход в программу, включите этот флажок.
  • Группы доступа пользователей - позволяет выбрать группы, настройки доступа которых распространяются на созданных пользователей.
