Создание грейдов

Обработка используется для создания грейдов и распределения имеющихся должностей (позиций) штатного расписания по ним. Используется балльно-факторный метод – оценка позиций компании по нескольким ключевым критериям и распределение по грейдам исходя из суммы баллов.

Критерии оценки позиций можно задать вручную по одноименной ссылке, или использовать уже существующие.

Для каждой должности (позиции) указываются веса критериев, сумма этих весов определяет суммарное количество баллов. По кнопке Распределить указанное количество грейдов распределяется по позициям. Наибольший по номеру грейд присваивается должности (позиции) с наибольшим количеством баллов, далее по убывающей. Если у должности(позиции) нет баллов, ей присваивается грейд 1. Количество грейдов по умолчанию определяется как количество позиций штатного расписания, деленное на 5 (но не меньше 2 и не больше 20). При необходимости количество грейдов и их распределение можно скорректировать вручную.

После распределения для создания грейдов в справочнике Грейды необходимо нажать Создать грейды.

В штатном расписании отобразятся распределенные грейды. После создания и распределения грейдов необходимо назначить их всем сотрудникам. Если работа в базе уже велась, это возможно сделать Кадровым переводом или документами, изменяющими оплату труда. Если работа в информационной базе не велась, то грейды назначатся сотрудникам при приеме на работу.

При приеме на работу, кадровых изменениях, изменениях оплаты труда при проведении документа будет произведен анализ: входят ли значения показателей, размеры начислений, указанные в документе в диапазон аналогичных показателей и размеров начислений, указанные в грейде, а также не введены ли запрещенные грейду начисления. Если это не так, будет выведено соответствующее сообщение.

Распределение позиций по гредйам можно увидеть в отчете Матрица грейдов. Проанализировать соответствие начислений и показателей грейдов и позиций штатного расписания можно в отчете Контроль грейдов.
