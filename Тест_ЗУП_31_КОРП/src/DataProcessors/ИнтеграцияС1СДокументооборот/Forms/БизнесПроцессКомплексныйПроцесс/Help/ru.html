Комплексный процесс состоит из набора запланированных действий (типовых процессов), которые могут стартовать, как по заранее настроенному порядку, так и при соблюдении определенных условий.

Вы можете создать новый комплексный процесс по заранее настроенным шаблоном. После старта процесса исполнители получают задачи, аналогичные тем, что они получают, например, после старта процесса Согласование.

Для дальнейшего повторного использования созданных процессов Вы можете сохранить их в качестве шаблонов.

Для контроля хода выполнения Вы можете назначить себя или любого другого пользователя, у которого есть права для работы с процессами и задачами, контролером комплексного процесса. Для контролера формируется отдельная задача, которую он будет выполнять одновременно с другими пользователями.
