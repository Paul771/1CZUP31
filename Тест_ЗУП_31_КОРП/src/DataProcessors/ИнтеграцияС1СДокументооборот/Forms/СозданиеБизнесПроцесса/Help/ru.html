При помощи помощника создания процесса можно выбрать подходящий под текущие нужды шаблон. Для удобства выбора все процессы рассортированы по 4 группам в порядке, рекомендованном для использования.

Рассмотрим эти группы подробнее.

Группа
Что содержит
Рекомендованные шаблоны Содержит уже настроенные шаблоны процессов для данного вида документов. Группа доступна, только если процесс создается на основании документа.
Общие шаблоны Содержит шаблоны процессов, которые не привязаны ни к одному из видов документов.
Новые процессы Содержит список для создания новых процессов с нуля, без использования шаблонов.В нем представлены либо все процессы, либо только те, которые подходят при создании на основании. Например, процесс Регистрация нельзя стартовать на основании контрагента или файла, поэтому в списке Новые процессы он представлен не будет.
Другие шаблоны Содержит шаблоны процессов, которые настраивает ответственный за нормативно-справочную информацию.
