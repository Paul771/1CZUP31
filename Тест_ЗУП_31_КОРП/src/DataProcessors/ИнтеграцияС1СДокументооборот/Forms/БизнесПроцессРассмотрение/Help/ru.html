Этот бизнес-процесс используется для автоматизации процесса рассмотрения документов руководителем. Сначала руководителю поступает задача "Рассмотреть", содержащая ссылку на документ. После того как руководитель рассмотрит документ и вынесет резолюцию, автору поступает задача "Обработать резолюцию". Из этой задачи автор имеет возможность на основе текстовой резолюции руководителя создать бизнес-процесс "Исполнение" или "Ознакомление".

Этот бизнес-процесс можно создавать непосредственно из карточки документа при помощи меню "Создать бизнес-процесс".

При выборе рассматривающего можно указать не только конкретного пользователя, но и роль.

Для просмотра текущего состояния бизнес-процесса (списка выполненных и не выполненных задач) используйте команду "Задачи" в панели навигации карточки документа, отправленного на рассмотрение.

Если не был выполнен запуск бизнес-процесса, заполнение формы может быть выполнено на основании шаблона бизнес-процесса. Для этого необходимо выполнить команду "Заполнить по шаблону" на командной панели формы. Если для данного типа бизнес-процесса и вида предмета бизнес-процесса настроены шаблоны бизнес-процессов, то система предложит выбрать один из шаблонов для заполнения формы. Если был настроен только один шаблон, то заполнение формы произойдет автоматически.
