В форме выводится информация о входящем, исходящем или внутреннем документе, соответствующем объекту внешней по отношению к 1С:Документооборот системы.

Внешнему объекту может соответствовать только один входящий, исходящий или внутренний документ в 1С:Документообороте.
В поле "Документ" выводится наименование, номер и дата этого документа.
Кнопка "Создать" позволяет создать новый документ, соответствующий внешнему объекту.
Кнопка "Очистить" очищает связь между текущим объектом и объектом 1С:Документооборота. 
Изменить или просмотреть уже созданный документ можно по гиперссылке на документ.
Кнопка "Выбрать..." позволяет привязать к внешнему объекту уже существующий в 1С:Документообороте документ.

В списке "Связанные документы" выводятся документы, связанные с документом, соответствующем внешнему объекту.
Например, для договора в этом списке могут отображаться дополнительные соглашения, акты, спецификации и пр.
Кнопка "Создать" позволяет создать новый связанный документ, при этом требуется указать тип связи.
Кнопка "Приложить" позволяет сделать связанным уже существующий документ.

В списке "Бизнес-процессы и задачи" выводится дерево связанных с документом бизнес-процессов и задач.
Кнопка "Создать" вызывает окно создания нового бизнес-процесса.
Кнопка "Открыть" открывает текущий элемент.

В списке "Файлы" выводятся файлы, связанные с документом.
Кнопка "Создать" вызывает окно создания нового файла.
В группе "ЭЦП", при использовании подсистемы работы с ЭЦП, доступны команды для работы с ЭЦП.

В списке "История переписки" выводится дерево переписки по документу.
Кнопка "Создать" вызывает окно создания нового письма.
Кнопка "Открыть" открывает текущий элемент.

В случае, если для текущего типа объекта не настроено заполнение объекта в 1С:Документообороте, об этом выводится предупреждение.
