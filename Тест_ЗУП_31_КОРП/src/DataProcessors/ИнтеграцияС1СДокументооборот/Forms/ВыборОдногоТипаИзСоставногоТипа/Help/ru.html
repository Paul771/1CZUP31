Выбор типа значения осуществляется, когда в поле на карточке объекта может быть указано значение нескольких типов. В таком случае сначала необходимо выбрать тип значения.

 

Для продолжения заполнения поля карточки выберите требуемый тип данных и нажмите кнопку "ОК".

 

При создании бизнес-процесса документооборота выберите тип бизнес-процесса и нажмите кнопку "ОК".
