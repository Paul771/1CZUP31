Для открытия карточки файла используйте клавишу F2 или кнопку  ("Изменить текущий элемент") командной панели списка файлов. 

Карточка файла содержит различные сведения о файле и кнопки для выполнения основных действий с файлом. Дополнительные команды для работы с файлом находятся в подменю "Все действия" командной панели формы файла.

Команда Описание
Открыть Открывает файл внешним приложением для просмотра.
Сохранить как Сохраняет файл в указанный каталог на жестком или сетевом диске. 
Обновить из файла на диске Обновляет файл в информационной базе выбранным файлом на жестком диске.
Подписать Открывает диалог для выбора сертификата и установки подписи.
Добавить ЭЦП из файла Добавляет выбранный файл подписи к подписям файла.
Сохранить вместе с ЭЦП Сохраняет файл в указанный каталог на жестком или сетевом диске, а также сохраняет подписи файла - все или выбранные, в зависимости от настройки "Действия при сохранении с ЭЦП"
Открыть подпись Открывает форму для просмотра информации о подписи: кому выдан сертификат, кто установил подпись, дату установки подписи, комментарий к подписи
Проверить Проверяет истинность выделенных подписей
Проверить все Проверяет истинность всех подписей
Сохранить Сохраняет выбранную подпись в файл
Удалить Удаляет выделенные подписи
