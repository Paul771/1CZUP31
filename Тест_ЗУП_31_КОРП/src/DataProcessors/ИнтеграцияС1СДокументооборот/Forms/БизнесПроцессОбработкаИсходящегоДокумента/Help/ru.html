Бизнес-процесс  "Обработка исходящего документа" состоит из цепочки простых бизнес-процессов, определенных как шаблоны: Согласование, Утверждение и Регистрация.

Если не был выполнен запуск бизнес-процесса, заполнение формы может быть выполнено на основании шаблона бизнес-процесса. Для этого необходимо выполнить команду "Заполнить по шаблону" на командной панели формы. Если для данного типа бизнес-процесса и вида предмета бизнес-процесса настроены шаблоны бизнес-процессов, то система предложит выбрать один из шаблонов для заполнения формы. Если был настроен только один шаблон, то заполнение формы произойдет автоматически.

В поле Наименование указывается наименование составного бизнес-процесса. Если бизнес-процесс создается по шаблону и в шаблоне задано наименование, то оно переносится в поле Наименование.

Поле Предмет заполняется автоматически на основании данных документа, отправляемого на обработку.

В поле Автор указывается автор создаваемого бизнес-процесса (при создании будет указан текущий пользователь).

В поле Согласование задается шаблон бизнес-процесса Согласование.

В поле Утверждение задается шаблон бизнес-процесса Утверждение.

В поле Регистрация задается шаблон бизнес-процесса Регистрация.

С помощью поля Главная задача указывается, что данный бизнес-процесс создан в рамках вышестоящего бизнес-процесса.
