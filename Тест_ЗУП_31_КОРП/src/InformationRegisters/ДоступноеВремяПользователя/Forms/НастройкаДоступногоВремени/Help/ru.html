Настройка доступного времени позволяет указать расписание, по которому пользователь доступен для планирования.

Если установлена настройка "Доступен всегда", то пользователь будет считаться доступным всегда, кроме конкретного времени, указанного как занятое, и времени, занятого записями календаря.

Если установлена настройка "Занят всегда", то пользователь будет считаться занятым всегда, кроме конкретного времени, указанного как доступное.

Если установлена настройка "Доступен по расписанию", то пользователь будет считаться доступным в соответствии с указанным расписанием.

Помимо данной настройки, на доступность пользователя влияют записи календаря и отдельные записи занятости, указывающие на доступность пользователя в конкретное время.
