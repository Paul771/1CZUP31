Анализ целевых показателей сотрудников

Отчет предназначен для сравнения значений целевых показателей сотрудников за указанный период. Данные в отчет подбираются из рабочего места расчет показателей эффективности.
Возможен быстрый отбор по целевому показателю, сотруднику, подразделению и должности.

Плановые и фактические значения целевых показателей сотрудников

Отчет предназначен для анализа изменения значений целевых показателей сотрудника за произвольный период. Горизонтальными линиями обозначены значения по уровням целевых показателей.
Данные в отчет подбираются из рабочего места расчет показателей эффективности.
Возможен быстрый отбор по сотруднику и целевому показателю.
